# SearchApp
React-Native application for searching images online
Application uses Bing search API

## Bing Search API:
https://azure.microsoft.com/en-us/try/cognitive-services/?api=bing-web-search-api

## Aplication file structure:
 -- src/  
 ----components/  
 -------App.js  
 -------SearchResults.js     
 ----images/  
 -------backBtn.png    
 ----ajax.json  
 --index.js
